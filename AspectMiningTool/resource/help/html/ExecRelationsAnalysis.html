Execution Relation Analysis

Running Execution Relation Analysis

In order to perform a Execution Relation Analysis of a Java Project, please follow the steps listed below:

  1. Right-click on the Java Project of interest
  2. Select Perform Mining -> Execution Relation Analysis

Filter Seeds

In order to filter the Execution Relation Analysis results, please follow the steps listed below:

  1. Click on View Filter
  2. Select the desired Threshold Value
  3. Checked the desire Getter/Setter option
  4. Click on the OK Button

Saving Results

In order to save the Execution Relation Analysis results, please follow the steps listed below:

  1. Click on the Save image of the View
  2. Select the save location

Selection of a confirmed Seed

In order to confirm a Seed, please follow the steps listed below:

  1. Right-click on the method of interest
  2. Select "Select as Seed" option

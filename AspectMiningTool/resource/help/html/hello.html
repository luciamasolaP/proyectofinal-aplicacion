Synergy Analysis User's Guide

This document contains the user- and installation-guide for Synergy Analysis Tool (SAT). SAT is an aspect mining tool that supports 5 techniques to identify crosscutting concerns in a source code. This techniques are:

  • Fan-in analysis
  • Unique Methods Analysis
  • Execution Relation Analysis
  • Redirections finder Analysis
  • Synergy Analysis

Install

Save the SAT.jar distribution in the plugins directory of your Eclipse installation and (re-)start Eclipse.

Requirements

    Eclipse 3.4 or higher

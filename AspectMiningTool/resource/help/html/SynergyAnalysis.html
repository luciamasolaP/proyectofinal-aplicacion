Synergy Analysis

Running Synergy Analysis

In order to perform a Synergy Analysis of a Java Project, please follow the steps listed below:

  1. Right-click on the Java Project of interest in the Navigator View
  2. Select Perform Mining -> Synergy Analysis
  3. Select the parameters and Click on the OK button

Saving Results

In order to save the Synergy Analysis results, please follow the steps listed below:

  1. Click on the Save image of the View
  2. Select the save location

Selection of a confirmed Seed

In order to confirm a Seed, please follow the steps listed below:

  1. Right-click on the method of interest
  2. Select "Select as Seed" option

Edit Seeds Information

In order to a edit the confirmed Seeds Information, please follow the steps listed below:

  1. Click on the Seed of interest and edit the seed description
  2. On the Right Table, select whether the Related Method applies or not
  3. On the Right Table, edit the Related Method description

Saving Results

In order to save the Execution Relation Analysis results, please follow the steps listed below:

  1. Click on the Save image of the View
  2. Select the save location

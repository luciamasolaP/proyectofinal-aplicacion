Redirector Finder Analysis

Running Redirector Finder Analysis

In order to perform a Redirector Finder Analysis of a Java Project, please follow the steps listed below:

  1. Right-click on the Java Project of interest in the Navigator View
  2. Select Perform Mining -> Redirector Finder Analysis

Filter Seeds

In order to filter the Redirector Finder Analysis results, please follow the steps listed below:

  1. Click on View Filter
  2. Select the desired Threshold Value
  3. Checked the desire Getter/Setter option
  4. Click on the OK Button

Saving Results

In order to save the Redirector Finder Analysis results, please follow the steps listed below:

  1. Click on the Save image of the View
  2. Select the save location

Selection of a confirmed Seed

In order to confirm a Seed, please follow the steps listed below:

  1. Right-click on the Class of interest
  2. Select "Select as Seed" option
